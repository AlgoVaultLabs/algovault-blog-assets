# algovault-blog-assets

Public CDN-source for AlgoVault editorial blog (Hashnode at blog.algovault.com)
images. Auto-generated by the editorial pipeline at `204.168.185.24` (per
`/opt/algovault-editorial/lib/screenshot_generator.mjs`).

Layout: `<post-slug>/{cover,api-response,agent-loop}.png` (1200×630 or wider).

Public repo so raw.githubusercontent.com URLs resolve unauthenticated for
Hashnode's CDN to fetch + cache.
